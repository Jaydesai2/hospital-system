# 17. Responsive Layout Rules

To avoid UI breakage across devices.

---

## Grid Layout

```html
grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-6
```

Meaning:

| Screen  | Columns |
| ------- | ------- |
| Mobile  | 1       |
| Tablet  | 2       |
| Desktop | 4       |

---

## Responsive Container

```html
max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
```

This keeps layout centered.

# 19. Accessibility Rules (Very Important)

Healthcare apps **must be accessible**.

Rules:

1️⃣ Minimum contrast ratio **4.5:1**

2️⃣ Buttons must have **hover + focus states**

3️⃣ Inputs must have **clear labels**

4️⃣ Avoid tiny fonts (<14px)
